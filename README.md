# i1810b！  
### プログラマカレッジ・チーム製作（7名）  
　  
### i1810b！ECサイトURL（公開時間帯：平日10時 ～ 18時）  
##  http://www.internousdev.com/i1810b/HomeAction

#### 概要
日本の伝統工芸品を購入できるECサイト

#### 主な機能
1．ユーザー登録
2．ログイン・ログアウト
3．カート
4．商品一覧
5．商品検索
6．マイページ
7．購入履歴一覧・削除
8．パスワード再設定

#### 開発期間
2018/10/01~2018/10/31(21営業日)

#### 開発環境
OS: Mac OSX 10.11.6
統合開発環境: Eclipse 4.6.3
